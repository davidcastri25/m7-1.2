Creación de cohetes

Para crear un cohete es necesario asignarle un código, que debe estar formado por 8 caracteres (pueden ser letras o números).

Cada cohete necesitará propulsores para moverse: debemos indicar en el formulario la potencia máxima de cada propulsor del cohete. La potencia de un propulsor se indica a través de múltiplos de 10 (potencia mínima 10, potencia máxima 100).

Un cohete necesita un mínimo de un propulsor (el propulsor 1) y no puede tener más de 6 propulsores.

Funcionamiento de la aplicación web

Podrás usar el formulario para crear cohetes. Para poder iniciar la partida debe haber un mínimo de 2 cohetes creados, mientras que el número máximo de cohetes será de 99999999.

Una vez generados los cohetes que desees, puedes iniciar la aplicación mediante el botón de iniciar partida. Este botón hará aparecer el menú de cada cohete con las opciones que se pueden aplicar a cada uno, además de un botón para que aparezcan por pantalla los datos de todos los cohetes.
